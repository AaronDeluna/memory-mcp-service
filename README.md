# memory-mcp

`memory-mcp` - сервис локальной памяти для СТП.

Сервис индексирует Markdown-файлы, сохраняет документы и чанки в PostgreSQL, строит полнотекстовый и семантический поиск, а также предоставляет API для других сервисов системы: `orchestrator`, `researcher`, `idea-agent`, `notifier`.

## Назначение

`memory-mcp` отвечает за хранение и поиск накопленных знаний.
